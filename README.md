# Flutter Unsplash Client

A simple Flutter app showing trending images from [Unsplash](https://unsplash.com/)

## Getting Started

## Provide an Api Key:
create new file `Keys.dart` containing your api key:
```dart
class Keys {
  /* unsplash api key */
  static String UNSPLASH_API_CLIENT_ID = 'YOUR_API_KEY';
}
```
